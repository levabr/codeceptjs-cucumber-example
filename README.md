CodeceptJS example: Cucumber + Allure
========================

This is a sample project demonstrating how to use CodeceptJS with Cucumber and Allure.

### Run tests

To run tests in chromium browser, type from command line:

```
npx codeceptjs run
```

### See a report

Run the following command:
```
allure serve output
```
